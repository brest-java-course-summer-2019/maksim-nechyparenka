HTML (от англ. HyperText Markup Language — «язык гипертекстовой разметки»)

— стандартизированный язык разметки документов во Всемирной паутине. Большинство веб-страниц содержат описание разметки на языке HTML (или XHTML).

  1. Язык HTML интерпретируется браузерами; полученный в результате интерпретации форматированный текст отображается на экране монитора компьютера
  2. или мобильного устройства.
